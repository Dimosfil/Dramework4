# Dramework4

Dramework4 is a Unity runtime framework/package for dependency injection,
lifecycle orchestration, messaging, and testing utilities used by HappyCoder
projects.

## Unity Version

- Unity `2022.3.0f1` or newer compatible `2022.3` release.

## Package

The Unity package manifest is located at:

```text
Assets/IG/package.json
```

Package name:

```text
ru.indiega.happycoder.dramework4
```

Runtime source:

```text
Assets/IG/HappyCoder/Plugins/Dramework4/Code/Runtime
```

EditMode tests:

```text
Assets/IG/HappyCoder/Plugins/Dramework4/Code/Tests/EditMode
```

## Development

Open the repository root as a Unity project. Unity and IDE generated files such
as `Library/`, `Temp/`, `UserSettings/`, `*.csproj`, and `*.sln` are ignored.
